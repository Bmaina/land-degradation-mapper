## How to get these images

1. Run `LandDegMapper_Colab_Fixed.ipynb` through Step 12
2. Find the files in Google Drive at:
   `/MyDrive/LandDegMapper/outputs/`
3. Download them and place them in this folder
4. Run:
   ```bash
   git add docs/images/
   git commit -m "Add output screenshots"
   git push
   ```
